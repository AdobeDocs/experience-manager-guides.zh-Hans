# 使用FrameMaker {#id221BH0L03XA}

FrameMaker支持在AEM Guides中创作和发布内容。 FrameMaker附带一个到Adobe Experience Manager的现成连接器。 FrameMaker还为您提供了一个易于使用的界面，允许您在分布式协作环境中维护文档的版本。 您可以在FrameMaker中创作内容并以不同格式发布 — PDF、HTML5、EPUB和DITA。 您还可以执行各种文件管理操作，如签出、带依赖项签出、签入、刷新等。 有关详细信息，请参阅[FrameMaker帮助](https://help.adobe.com/zh_CN/framemaker/using/index.html)。
