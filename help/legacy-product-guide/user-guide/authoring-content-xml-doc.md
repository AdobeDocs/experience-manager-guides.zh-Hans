# 使用AEM Guides创作内容 {#id221EB0P0E20}

AEM Guides附带三个现成的编辑器。 您可以使用它们来创建、编辑和发布文档。

1. **Web编辑器**：使用它创建和编辑结构化文档。
1. **映射编辑器**：使用它创建和编辑DITA映射和bookmap文件。
1. **Ditaval编辑器**：使用它生成条件输出。
