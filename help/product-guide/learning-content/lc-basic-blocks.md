# 将基本构建块添加到主题

为了更清楚地了解如何创建学习主题并向其中添加基本构建块，以下视频简要介绍了可用的功能。


>[!VIDEO](https://video.tv.adobe.com/v/3469535/learning-content-aem-guides)


您可以使用编辑器工具栏上提供的基本编辑功能，如下所述：

- **插入选项**：提供用于添加[添加交互式小组件](./lc-widgets.md)以及[结构元素](./lc-other-insert-options.md)的选项，这些小组件包括可折叠项、轮盘、热点、选项卡、翻转卡和单击显示等，这些元素包括Iframe、块引号、代码块等。 使用此菜单向学习内容添加功能和多样性，使其富有吸引力且结构合理。

  ![](assets/insert-menu-options-lc.png){width="650" align="left"}

- **文本组件**：为您的内容添加标题/标题、段落、内嵌引号、上标、下标和引文。

  ![](assets/text-learning-content.png){width="650" align="left"}

- **未排序列表**：在内容中添加未排序列表。

  ![](assets/unordered-list.png){width="650" align="left"}

- **排序列表**：在内容中插入编号列表。

  ![](assets/ordered-list.png){width="650" align="left"}

- **表**：将包含所需维度的表插入到内容中。 您可以使用&#x200B;**内容属性**&#x200B;面板进一步管理各种表属性，如下所示。

  ![](assets/table-learning-content.png){width="650" align="left"}

- **图像**：将图像与替换文本和屏幕提示一起插入到内容中。 可以从存储库或通过外部URL添加图像。 此外，可以使用&#x200B;**内容属性**&#x200B;面板修改图像属性。

  ![](assets/image-learning-content.png){width="650" align="left"}

- **多媒体**：将视频和音频添加到内容中。 您可以使用&#x200B;**内容属性**&#x200B;面板自定义其属性。

  ![](assets/video-learning-content.png){width="650" align="left"}

- **可重复使用的内容**：允许您合并资源或存储库中的现有内容以便可重复使用。 执行以下步骤以插入可重用内容：

   1. 从工具栏中选择&#x200B;**可重用的内容**。
将打开**重用内容**&#x200B;对话框。
   2. 导航并选择所需主题以将其内容包含在当前课程中。
   3. 选择要添加的内容的ID；预览将显示在旁边以供参考。

      ![](assets/reuse-content-dialog.png){width="650" align="left"}

   4. 选择&#x200B;**选择**。

  插入内容。 例如，有关车辆结构的部分是一个内容片段，将被重复使用并添加到课程的主题中。 该类型显示为&#x200B;**引用**，其&#x200B;**ID**&#x200B;反映在&#x200B;**内容属性**&#x200B;面板中。

  ![](assets/reusable-content.png){width="650" align="left"}


- **符号**：将您选择的符号添加到列表中的内容，如下所示。

  ![](assets/symbol-learning-content.png){width="350" align="left"}


- **超链接**：将超链接添加到内容中的所需位置。 它可以是文件引用、Web URL或电子邮件链接，如下所示。

  ![](assets/hyperlink-learning-content.png){width="650" align="left"}

此外，**菜单**&#x200B;下拉菜单提供对编辑操作（“剪切”、“复制”、“删除”）、“查找和替换”和“版本”标签的访问权限。
