# 配置PDF输出预设

创建预设后，配置PDF预设设置。 预设配置选项组织在“常规”、“元数据”、“布局”、“安全性”、“打印”和“高级”选项卡下。 有关详细信息，请查看[PDF输出预设配置](../web-editor/native-pdf-web-editor.md)。
