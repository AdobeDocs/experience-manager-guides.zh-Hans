# 编辑器简介 {#id181OD0QE05Z}

Experience Manager Guides附带一个易于使用的编辑器，用于创建和编辑结构化文档。 该编辑器对作者隐藏了DITA结构的所有复杂内容。 编辑器提供了用户通常需要在文档中工作的DITA元素列表。

## 新编辑器

>[!NOTE]
>
> 从2026.05.0版本开始，Experience Manager Guides as a Cloud Service中提供了新编辑器。 请联系您的客户成功团队以启用此功能。

新编辑器在创作体验方面有了显着改进，在性能、可用性和UI/UX方面有了重点改进。 它专为DITA映射和DITA主题而设计，支持高效地创作大型、复杂和高度结构化的内容，同时保持完全向后兼容并且不影响非DITA内容。

### 主要功能

- 大型文件的性能改进：通过支持撤消/重做和脏标记，增强了大型DITA文件的响应能力，从而确保可靠的更改跟踪而不会降低性能。
- 现代化的表创作：通过拖放重新排序、上下文工具栏以及批量行或列插入来提供更直观的表编辑体验，以实现更快、更准确的表管理。
- 增强的创作模式：提供直接在创作模式下可配置的XML注释和属性可见性，从而减少在处理结构化内容时切换到Source视图的需求。
- 高级编辑模式和工具：为主题引入并排创作和Source模式以及快速插入菜单，只需很少的上下文切换即可实现更快的结构化创作。
- UI和UX细化：在创作和预览模式下通过深色主题和更丰富的默认CSS提高了可读性和视觉舒适性，并提供更现代的编辑界面。
- 集中式编辑器设置：提供一个统一的设置面板来个性化创作体验，包括对标记、属性、XML注释、不间断空格和编辑器中的快速插入菜单的控制。

### 新编辑器的限制

新编辑器当前不支持以下功能。 计划在即将发布的版本中引入这些功能。

- 在启用了跟踪更改的情况下创作（任何现有的跟踪更改都会保留，并被视为常规内容）
- 创作视图中的图像大小调整器


>[!NOTE]
>
>新编辑器不支持DITA映射的布局视图。


## 旧编辑器

Experience Manager Guides中的旧编辑器为结构化内容提供了简单而可靠的创作体验，抽象DITA的复杂性，同时确保遵守受支持的标准，如DITA 1.2、1.3、轻量DITA和专用DITA。 它提供了使用通用块和内联元素的基本工具，使其非常适合于在结构化框架中一致且受控的内容创建。
