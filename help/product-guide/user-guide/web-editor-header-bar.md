# 编辑器中的标题栏

标题栏是显示Adobe Experience Manager徽标的编辑器的顶部栏（如果您将Unified Shell用作Experience Manager Guides UI，则为Unified Shell）。 当您选择徽标时，它会将您定向到Experience Manager导航页面。

>[!BEGINTABS]

>[!TAB 新编辑器]

此视图显示内容在新编辑器中的呈现方式

![](./images/web-editor-header-bar-editor-2-0.png)

>[!TAB 旧编辑器]

此视图显示内容在旧编辑器中呈现的方式

![](./images/web-editor-header-bar.png)

>[!ENDTABS]

使用工具栏中的&#x200B;**展开**&#x200B;图标隐藏标题栏并最大化内容区域。 要恢复标准视图，请选择&#x200B;**退出扩展视图**。

![](./images/web-editor-header-bar-expand-option.png){width="350"}
