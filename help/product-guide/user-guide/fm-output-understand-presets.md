# 了解输出预设 {#understanding_output_settings}

AEM Guides支持为PDF、HTML5和EPUB格式的FrameMaker文档创建输出。 使用这些输出格式，您可以配置各种输出预设。 输出预设表示您希望发布内容的自定义输出格式。

以下各节将介绍适用于支持的输出格式的选项。
