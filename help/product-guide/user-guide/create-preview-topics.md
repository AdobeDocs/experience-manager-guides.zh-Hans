# 创建和预览主题 {#id21BQE0RJ042}

AEM Guides附带了一项强大的功能，可从Assets UI创建DITA主题。 然后，您可以从Assets UI预览主题。 预览模式还提供了许多强大的功能，可帮助您执行关键操作，如查看元数据、在Web编辑器中编辑主题、打开源代码以及查找基于Web的版本差异。
