# 自定义编辑器 {#id181AI0003PN}

AEM Guides附带一个功能强大的编辑器，通过该编辑器，您的作者可以创建和编辑DITA文档。 您可以自定义编辑器的工具栏，以添加或删除您可以从工具栏访问的任何功能。 此外，您还可以配置文件的自动保存、在元素插入到文档中时为其生成ID等等。
