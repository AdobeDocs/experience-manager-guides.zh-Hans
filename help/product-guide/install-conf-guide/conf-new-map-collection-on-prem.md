# 为内部部署配置新映射收藏集

以下步骤说明了如何在On-Premise环境中启用New map收藏集。

1. 打开Adobe Experience Manager Web控制台配置页面。

   用于访问配置页面的默认URL为：

   ```http
   http://<server name>:<port>/system/console/configMgr
   ```

1. 搜索并选择&#x200B;**com.adobe.fmdita.config.ConfigManager**&#x200B;包。

1. 启用设置&#x200B;**启用新映射集合** (enable.new.map.collections)。

   ![](assets/new-map-collection.png){width="800"}

1. 选择&#x200B;**保存**。
