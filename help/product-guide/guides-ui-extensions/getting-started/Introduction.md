# AEM Guides扩展包

AEM Guides扩展包允许您根据需要自定义应用程序。 是否要在工具栏中添加其他按钮？ 注释框中的新输入字段？ 上下文菜单中的另一个选项？ 现在你可以全部做了。

前往 [安装指南](./integrating-customisations.md) 以了解如何使用此包

> **_注意：_** AEM Guides版本>= 4.3（内部部署）和>= 2310（云）支持此AEM Guides扩展包
