# AEM Guides中的报表

在组织设置中，您需要先验证内容的整体完整性，然后再推送文档供最终用户使用。 在多用户和大型环境中，这种需求甚至更加重要，因为内容可以在不同的上下文中重复使用，因此验证内容或文档是必要的。 在结构化内容的上下文中，通常需要一些特定类型的报表。


## 会话概述

在本次研讨会中，我们向用户展示了现成的报告。 我们大致涵盖了以下内容：
- 文档（地图）和主题报表
- 元数据报表和批量元数据管理选项
- 内容重用报表
- 内容导入正确性
- 利用AEM Assets报告功能
