# 本机PDF发布要点

AEM Guides具有内置的PDF发布引擎，该引擎为用户提供了以PDF格式创建、开发和发布内容的功能。 其丰富的功能允许创建各种页面布局、自定义样式表以及设计无缝内容发布所需的各种PDF组件。

## 会话概述

在此会话中，您将了解如何使用AEM Guides中的本机PDF引擎发布PDF
