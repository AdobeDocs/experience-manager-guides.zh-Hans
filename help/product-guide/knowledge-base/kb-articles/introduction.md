# Experience Manager Guides知识库

知识库包含适用于Experience Manager Guides的实用的场景驱动文章，在核心产品文档之外涵盖常见配置、自定义和用例。

| 类别 | 描述 |
|---|---|
| 创作 | 在编辑器中创建和重用内容的提示和技术，包括编辑器自定义、元数据管理和内容验证。 |
| Guides扩展框架 | 了解如何自定义Experience Manager Guides UI，从扩展框架快速入门到构建自定义组件、小部件和查看应用程序自定义项。 |
| 外部数据Source | 关于将外部数据源连接器与Experience Manager Guides集成和配置的指南。 |
| 发布 | 生成输出的最佳实践，包括本机PDF、DITA-OT、AEM Sites模板和基于微服务的发布配置。 |
| 工作流 | 配置并使用后期生成工作流在生成输出后自动执行步骤。 |
| 翻译 | 有关如何提高翻译性能并管理在翻译中包括或排除哪些内容的说明。 |
| 用例 | 了解现实世界的情况，如条件内容、使用键值以及已发布输出中的上下文内容变量。 |
