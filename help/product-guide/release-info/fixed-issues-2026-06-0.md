# 修复了2026.06.0版本中的问题

本文介绍Adobe Experience Manager Guides as a Cloud Service 2026.06.0版本中修复的各个方面的错误。

有关新增功能和增强功能的详细信息，请查看[2026.06.0版本中的新增功能](whats-new-2026-06-0.md)。

了解2026.06.0版本[&#128279;](upgrade-instructions-2026-06-0.md)的升级说明。

## 创作

- 在使用基于单位的大小（如`in`、`mm`或`px`）在图像属性对话框中的&#x200B;**宽度**&#x200B;和&#x200B;**高度**&#x200B;字段之间切换焦点时，值将递增而不是保持稳定。 （指南 — 45929）

## 编辑器2.0

- 在`<keydef>`或`<topicref>`内的`<topicmeta>`中复制并粘贴`<keywords>`会导致关键字被插入不需要的外部标记中。 （指南 — 45800）
- 使用单位（如`mm`）指定的图像尺寸无法正确呈现，导致图像以其原始大小而不是指定的尺寸显示。 （指南 — 46275）
- 在启用了标记视图的情况下执行拖放操作时，选择内容以及部分XML或DITA标记会留下不需要的孤立标记，从而导致不正确的内容或视图。 （指南 — 28191）
- 在表格的“标签”视图中，当光标位于折叠条目标签正下方的单元格时，按向上箭头键会跳过折叠的标签，并将光标移动到文档的起始位置。 （指南 — 45408）
- 从表上下文工具栏执行任何操作都会意外关闭工具栏，中断后续的表操作。 （指南 — 45405）
- 从大纲视图或痕迹导航使用&#x200B;**此项后插入**&#x200B;或&#x200B;**此项前插入**&#x200B;后，光标将移动到任意位置而不是新添加的标记内。 （指南 — 45147）
- 当&#x200B;**编辑MathML**&#x200B;选项在只读模式下显示不正确或文件被其他用户签出时，它允许用户更新MathML内容，即使文件不应可编辑。 （指南 — 45172）
- 当使用退格键删除XML注释时，在删除注释标记的内容后不会删除该注释标记，并且删除操作将继续到前面的元素中。 （指南 — 45240）
- 对图像使用&#x200B;**复制路径**&#x200B;或&#x200B;**复制UUID**&#x200B;选项时，将返回完整的演绎版路径而不是原始资源路径。 （指南 — 45278）
- 将同一主题中的内容复制粘贴到编辑器中的无效位置会插入意外的`<foreign>`标记。 （指南 — 45378）
- 在Windows上，复制和粘贴表行会将意外属性（如`data-pm-slice`）添加到表元素，从而导致阻止保存文档的标记错误。 （指南 — 45784）
- 在映射或主题中使用上下文菜单中的&#x200B;**复制**&#x200B;选项，然后粘贴内容，可在粘贴的输出中插入额外的`<data>`标记。 （指南 — 45703）
- 从`cmd`元素拖动部分选定内容时，无法在现有文本之间或另一个`cmd`元素的末尾放置该内容。 （指南 — 44883）
- 将`scale`属性应用于表时，该表在创作和预览模式下无法以配置的大小呈现。 （指南 — 45984）
- 粘贴从外部源（如“绘制”或“剪切工具”）复制的图像，不会将图像插入到主题中。 （指南 — 45983）
- 保存后，主题标题中使用的`keyref`不会出现在目录或主题选项卡中，即使在浏览器刷新后也是如此。 （指南 — 45799）
- 使用对注释的版本进行并排视图在编辑器中打开审阅任务，会将版本显示为“无”，而不是主题的关联版本。 （指南 — 45127）
- 访问“审阅”任务页面时，不会显示主题之间的分页符，从而导致内容部分的连续渲染。 （指南 — 46777）
- 审核页面样式与新编辑器体验不一致，从而导致可视化体验不一致(GUIDES-46061)

## 资产管理

- 当映射包含指向非DITA资源（如`.html`）的外部`topicref`时，其预览未显示在Assets UI中。 （指南 — 45409）
- 当选择大量源文件夹时，“批量移动工具”对话框溢出，缺少滚动条，导致仅使用鼠标的用户无法访问目标路径字段和操作按钮。 （指南 — 45805）
- 从Assets UI复制资源时，我们收到`DXML reprocessing failure`的通知。 此通知具有误导性，并由于复制成功而造成收件箱混乱。 （指南 — 45012）
- 在文件夹的Assets左边栏中打开“筛选器”面板时，搜索字段和多方面将保持禁用状态，直到面板关闭并重新打开为止。 （指南 — 42652）
- 在翻译工作流期间创建的内容片段和中间翻译资产会由计划的资产处理作业无意中处理，从而导致日志中出现异常，并对尚未准备就绪的资产进行错误处理。 （指南 — 42582）

## 发布

- 在PDF模板中使用`.plt`和`.css`文件时，右键单击上下文菜单中提供&#x200B;**生成ID**&#x200B;选项，尽管该选项不适用于这些文件类型。 （指南 — 45254）
- 在刷新浏览器后重新打开新创建的本机AEM Sites预设时，**保存**&#x200B;按钮默认未正确启用，即使未进行任何更改也是如此。 （指南 — 45171）
- 在复制标记为默认的Native PDF预设时，也会将复制的预设设置为默认预设。 一次只能将一个预设指定为默认值。 （指南 — 44786）
- `outputclass`属性不会传播到生成的HTML或本机PDF输出，从而阻止应用于MathML公式的自定义类影响渲染的输出。 （指南 — 44393）
- 批量激活使用新的AEM Sites模板生成的AEM Sites输出失败，导致无法成功复制到发布实例。 （指南 — 44049）
- 已识别出与DITA-OT包中的AEM Guides捆绑在一起的易受攻击的`jackson-databind`个JAR（版本2.9.8）。 （指南 — 43081）
- 从映射收集打开任何本机AEM Sites输出都会导致错误，声明未找到资源，从而阻止访问生成的输出。 （指南 — 42065）
- DITA映射中的`<reltable>`元素在本地PDF生成期间被忽略，导致输出中缺少“相关概念”、“相关任务”和类似的自动生成的交叉引用部分。 （指南 — 38333）
- 在AEM Sites上发布包含`processing-role=resource-only`个元素的DITA映射（使用旧版组件映射）时，会在其他方案（如`topicgroup`元素和特定内容配置）中为这些元素生成孤立网站页面。 （指南 — 37650）
- 将长名称的映射添加到映射收藏集时，映射收藏集UI将以扭曲的布局呈现。 此问题已在新映射收藏集中得到解决。 （指南 — 42062）
- 使用本机PDF引擎v1发布：
   - 在为某些内容生成本机PDF输出时，尽管中间HTML包含多个页面中的完整内容，但PDF中仅呈现第一页。 （指南 — 28270）
   - 启用了辅助功能设置的本机PDF输出中的内容读取顺序不正确。 页脚中的页码在主内容之前而不是结尾处读取。 （指南 — 27790）
   - 当自定义页面大小时，原生PDF输出中的颜色条不会拉伸整个页面宽度，并且会重叠，从而导致隐藏某些颜色框。 （指南 — 15505）
   - 本机PDF输出中不允许CSS `:is()`伪类选择器，从而导致与浏览器呈现的样式不同。 （指南 — 11328）

  >[!NOTE]
  >
  > 上述问题已在本机PDF引擎v2中得到解决。 有关详细信息，请查看[使用Native PDF引擎v2](../native-pdf/new-pdf-engine.md)。

## 翻译

- 将基线应用于具有多个资源的映射会延迟加载所选语言的翻译报表，有时会导致报表渲染前的请求超时。 （指南 — 45508）
- 当所选语言文件夹使用连字符区域设置代码（例如，`da-DK`）而不是下划线格式(`da_dk`)时，Workspace设置下的“翻译”选项卡中的语言组不会保留。 （指南 — 37843）

## 审阅

- 在审核UI的与主题名称相邻的左侧面板中缺少&#x200B;**版本历史记录**&#x200B;图标的工具提示。 （指南 — 45511）
- 在编辑活动审阅任务时，如果移除已属于审阅内容的主题，然后在不选择&#x200B;**更新**&#x200B;的情况下重新添加，则该主题的审阅者信息将丢失。 （指南 — 38774）
- 审核UI中的内容窗格在某些屏幕分辨率上显示水平滚动条，包括推荐的1600px分辨率，当内容超出窗格宽度时，会导致隐藏审核注释。 （指南 — 44082）

## 学习内容

- 使用从问题库插入选项向测验添加问题时，尽管问题ID有效，但简短答案问题未列出。 （指南 — 44942）
- 使用“从题库插入”选项插入问题时，打开时“从题库插入”对话框会闪烁或意外调整大小。 （指南 — 45524）
- 使用“从题库插入”选项插入问题时，如果问题标题包含图像，则会出现错误。 （指南 — 45383）
- 修改模板标题时，选择SCORM输出模板会导致应用程序错误。 （指南 — 45521）
- 子标题(`.vtt`)文件上传到文件夹后，在存储库或资源管理器视图中不可见。 （指南 — 46014）
- 在HTML预览中加载视频内容时，源(`src`)路径包含附加的演绎版路径，而不是保留原始文件路径，这会阻止视频正确呈现。 （指南 — 41776）
- 从Safari浏览器发布SCORM输出会导致将包下载为文件夹而不是zip文件，以及生成的内容中的呈现和功能问题（例如，拉伸内容、不可折叠内容等）。 （指南 — 45119）
- 在启用课程的“下一步”按钮之前会出现延迟，从而影响学习者导航体验。 （指南 — 45113）
- 尽管在预览中正确显示，但在发布输出时，简短回答问题样式呈现不正确。 （指南 — 46478）
- 在为包含视频的课程生成PDF输出时，不会呈现视频内容，并且只显示文件路径，而不是显示占位符图像，如视频缩略图或海报图像。 通过在原生PDF输出预设中启用&#x200B;**嵌入多媒体文件**&#x200B;选项，此问题已得到解决。(GUIDES-45117)
- CSS伪类和元素在指南的编辑器中以白色呈现，使它们不可见或难以在编辑器背景中读取。 （指南 — 45116）

## 已知问题

Adobe发现了2026.06.0版本的以下已知问题：

## 编辑器2.0

- 在Source和创作模式之间切换会导致内容不一致，主题的部分内容消失或在不同模式之间不显示。 （指南 — 47432）

- 在启用了&#x200B;**跟踪更改**&#x200B;的大纲视图中工作时，拒绝更改将删除标记中的全部内容，而非仅拒绝特定修改的内容。 （指南 — 48319）

- 使用文件夹配置文件中的`editor_toolbar.json`自定义编辑器工具栏时，预览模式中的&#x200B;**导出为PDF**&#x200B;按钮不执行任何操作。 （指南 — 47525）

- 执行删除操作时，可能会出现图像映射、结构化元素、内联格式标记以及不可合并的块之间光标移动和导航的一些细微不一致，有时会导致意外的光标或删除行为。 （指南 — 46756）

- 在映射编辑器的断开链接上使用`Ctrl+click`会触发应用程序错误。 （指南 — 45544）

- 在只读内容（如`conref`段落）之后紧接着的段落的开头按退格键可能会意外删除或合并可编辑的段落，从而导致可编辑段落意外删除。 （指南 — 45049）

- 将条件指示器应用于`bodydiv`等元素时，指示器会溢出到“完整标记视图”中的相邻标记中，从而导致不正确的可视渲染。 （指南 — 44971）

- 在编辑器中工作时，您无法选择`keyrefs`或从映射派生的键（包括在`term`或`abbreviated-form`元素中引用的映射中的术语条目），从而导致创作体验降级。 (GUIDES-45790) <br> **解决方法**：您可以使用属性值从右侧面板中更改`keyref`的值。

- 在“大纲”视图中，关闭并重新打开主题后重新启用&#x200B;**显示文本**&#x200B;不会将文本显示在元素标记旁边。 (GUIDES-48320) <br> **解决方法**：启用&#x200B;**显示文本**&#x200B;选项，然后重新打开主题。 重新打开后，文本将正确地显示在元素标记旁边。

- 使用&#x200B;**重命名元素**&#x200B;选项重命名内联标记时，痕迹导航不会立即更新，并且仅在光标移动到标记中或视图模式更改之后才会反映更改。 (GUIDES-44993) <br> **解决方法**：重命名内联标记后刷新浏览器以更新痕迹导航。

- 将MathML方程式作为`conref`插入时，无法正确呈现。 (GUIDES-46601) <br> **解决方法**：将MathML公式包装在`<p>`元素中，并将该`<p>`元素用作conref源。

## 审阅

- 当审核任务被重新分配给项目团队之外的用户时，尽管缺少项目成员资格，用户仍可以执行审核操作，而审核者可见性、审核状态跟踪和委派通知无法正常工作。 （指南 — 46602）

## 发布

- 当使用通过条件分析排除所有项目符号列表项的DITAVAL筛选条件发布内容时，输出错误地保留空的项目符号标记，而不是删除整个列表结构。 （指南 — 47238）

- 使用新基线发布原生AEM站点输出时，“主题列表”显示为空白，并且无法显示选定基线中包含的主题。 (GUIDES-46480) <br> **解决方法**：使用可通过配置管理器配置的旧基线发布本机AEM站点输出。
