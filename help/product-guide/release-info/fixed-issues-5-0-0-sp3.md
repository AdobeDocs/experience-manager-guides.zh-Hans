# 修复了5.0.0 Service Pack 3版本（2026年2月）中的问题


本文介绍Adobe Experience Manager Guides 5.0.0 Service Pack 3版本中修复的各个方面的错误。

了解5.0.0 Service Pack 3版本[的](upgrade-instructions-5-0-0-sp3.md)升级说明。

## 翻译

- 当最初作为具有特定版本（例如，在`/en/`下）的特定于语言的资产管理的图像移动到具有更新版本的全局文件夹并执行基线导出时，新基线将继续引用该图像的已过时的语言特定版本，从而导致基线导出失败。 (指南 — 39394)
