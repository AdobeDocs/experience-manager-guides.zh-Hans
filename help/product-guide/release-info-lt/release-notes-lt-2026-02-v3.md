# 2026年2月版产品培训和学习内容

本发行说明涵盖了产品培训和学习内容在2026年2月版中修复的新功能增强和问题。

## 新增功能增强

以下功能将在2026年2月发布的产品培训和学习内容中引入：

- **字幕支持**：您现在可以在&#x200B;**文件属性**&#x200B;中使用新的&#x200B;**添加字幕**&#x200B;选项向学习内容添加字幕。 这提高了清晰度并增强了课程内容的可搜索性。

  有关更多详细信息，请查看[为学习内容添加标题和子标题](../learning-content/lc-basic-blocks.md#add-title-and-subtitle-to-learning-content)。

  ![](assets/add-subtitles.png)

- **启用或禁用最终负面评分**：在配置测验属性时，可以使用&#x200B;**允许负面最终评分**&#x200B;选项来控制负面评分。 启用后，学习者即使应用了负标记，最终得分最低也为零。 这保证了学习者的积极性，确保分数永远保持在零以下。

  了解有关[测验属性](../learning-content/quiz-properties.md)的详细信息。

  ![](assets/negative-scores-lc.png)

- **通过右键单击删除小组件**：除了删除测验问题之外，您现在还可以通过&#x200B;**右键单击>删除项目**&#x200B;删除小组件，例如折叠卡、翻页卡和选项卡。 此增强功能将现有的&#x200B;*删除问题*&#x200B;功能扩展到小组件，允许您通过较少的点击次数和最少的导航删除问题。

  了解有关[使用交互式小组件](../learning-content/lc-widgets.md)的详细信息。

  ![](assets/delete-widget-items.png)
- **固定答案选择**：您现在可以固定特定的答案选择，以便它们的位置保持不变，即使在SCORM输出生成期间随机生成答案也是如此。 此选项对于诸如&#x200B;*以上所有*&#x200B;或&#x200B;*以上无*&#x200B;之类的选项特别有用。

  ![](assets/pin-question.png)
- **简短的答案类型**：简短的答案问题类型允许学习者使用简短的描述性字母数字答案进行回答，而不是选择预定义的选项。 此问题类型鼓励学习者用自己的语言积极回忆和表达自己的理解，使评估对学习者更有吸引力。

  了解有关[问题类型](../learning-content/quiz-insert-questions.md#question-types)的详细信息。


  ![](assets/short-answer.png)
- **测试问题的连续尝试**：您现在可以使用&#x200B;**学习者必须尝试每个问题才能继续SCORM输出预设中的**&#x200B;选项，对SCORM输出强制执行连续测试尝试。 启用后，学习者必须先回答每个问题，然后才能转到下一个问题，并且导航会受到限制，直到当前问题完成为止。 这可以确保引导式、逐步的评估流程和一致的学习体验。

  有关详细信息，请查看[配置SCORM输出预设](../learning-content/config-scorm-preset.md)。

  ![](assets/scorm-general-tab-v3.png)

## 修复的问题

以下问题已在产品培训和学习内容的2026年2月版中修复：

- 发布SCORM输出并将其部署到ALM时，L2测验报告显示使用多次尝试和随机选择问题库的测验总分和最高分有误。 (指南 — 38855)
- 在云服务器上生成任何课程时，由于`coralui3.css`样式表，版权页脚下会出现意外空格，导致布局不一致。 (指南 — 38853)
- 使用键盘导航带折叠的学习课程时，+符号或选项卡标题未高亮显示，导致无法直观识别活动元素。 (指南 — 38852)
- 对于使用SCORM炭模板或默认模板生成的课程，在横向模式的移动设备上访问时，目录（课程菜单）无法显示阻止导航的模块链接。 (指南 — 38851)
- 在Experience Manager Guides中复制课程的层级时，创建学习对象需要先创建学习组，因为不支持对象级别的添加。 (指南 — 38849)
- 尝试使用键盘访问“Match the following question type using the keyboard（匹配以下问题类型）”中的下拉选项失败，因为这些选项没有响应Tab键或箭头键而阻止导航。 (指南 — 38985)
- 应用标题样式预设会导致所选文本消失（可能是由于字体颜色变为白色），从而使文本不可选且不可见。 (指南 — 39981)
- 在Mozilla Firefox上使用Experience Manager Guides时，翻转卡片会在翻转后的背面以相反方式显示前端文本。 (指南 — 39983)
- 单击课程左窗格中的目录(TOC)后，即使测试失败，课程仍会显示完成状态。 (指南 — 40398)
- 尝试在ALM的测验中错误匹配以下问题类型，会导致选定选项未出现在报告中。 (指南 — 38640)
- 生成PDF输出时，不会保留应用的创作样式，从而导致设计不一致。(指南 — 38642)
